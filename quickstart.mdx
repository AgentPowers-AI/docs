## Install

<Tabs>
  <Tab title="Plugin (Recommended)">
    Install the full plugin with MCP tools, skills, and session hooks:
    ```bash
    claude plugin add AgentPowers-AI/agentpowers-plugin
    ```
    See [Plugin guide](/guides/plugin-usage) for full setup.
  </Tab>
  <Tab title="MCP Server">
    Add to your Claude Code MCP config:
    ```bash
    npx @agentpowers/mcp-server
    ```
    See [MCP Server guide](/guides/mcp-server) for full setup.
  </Tab>
  <Tab title="Skill (Zero Dependencies)">
    Add the standalone skill — no Node.js or Python required:
    ```bash
    claude skill add AgentPowers-AI/agentpowers-skill
    ```
    See [Skill guide](/guides/skill-usage) for details.
  </Tab>
  <Tab title="CLI (pip)">
    ```bash
    pip install agentpowers
    ```
  </Tab>
  <Tab title="npx (Zero Install)">
    ```bash
    npx @agentpowers/mcp-server search "code review"
    ```
  </Tab>
</Tabs>

## Search for Skills

```bash
ap search "code review"
```

This returns a table of matching skills with titles, prices, security status, and download counts.

## Install a Free Skill

```bash
ap install my-awesome-skill
```

The CLI downloads and extracts the skill to `~/.claude/skills/my-awesome-skill/`.

## Install a Paid Skill

```bash
ap install premium-agent
```

For paid skills, the CLI opens a Stripe Checkout page. After purchase, you receive a license code. The skill installs automatically after payment, or you can redeem the license code later:

```bash
ap install premium-agent --code XXXXXXXX-XXXXXXXX-XXXXXXXX
```

## Use In-Conversation (MCP Server)

The MCP server gives Claude access to marketplace tools during your conversation. Add it to your Claude Code settings or `.mcp.json`:

```json
{
  "mcpServers": {
    "agentpowers": {
      "command": "npx",
      "args": ["@agentpowers/mcp-server"]
    }
  }
}
```

See [MCP Server guide](/guides/mcp-server) for full setup details.

## Publish a Skill

<Steps>
  <Step title="Create your skill directory">
    ```
    my-skill/
    ├── SKILL.md      # Frontmatter with title, description, price
    └── ...            # Your skill files
    ```
  </Step>
  <Step title="Login">
    ```bash
    ap login
    ```
  </Step>
  <Step title="Publish">
    ```bash
    ap publish ./my-skill
    ```
    Your skill enters the security review pipeline. Once approved, it appears in the marketplace.
  </Step>
</Steps>
